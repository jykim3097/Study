display와 visibilty의 차이

**영어 뜻
display: 표시, 나타냄 -> 요소를 어떻게 표시할 지 선택
visibilty: 볼 수 있음 -> 요소를 보일지 말지 결정
**여기서 요소(즉 태그)는 inline, block
inline: 줄 속에 넣는 요소 <b><span><a> 등
         특정 문자열을 선택할 떄 사용하는 태그, 줄바꿈이 안됨
block: 좀 더 넓은 범위를 지정할 때 사용 <p><h1><div><ul><ol> 등
         가로 화면 100%차지, 자동으로 앞뒤 줄바꿈이 됨
display:none
-> 코드가 있는데도 아무것도 표현이 안됨

display:inline
-> 기본값으로 요소를 inline 요소처럼 표시, 앞뒤로 줄 바꿈 안됨

-> 요소를 block 요소처럼 표시, 요소 앞 뒤로 줄 바꿈이 됨
